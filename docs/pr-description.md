# feat(sync): complete advanced shared map controls

## 修改目标

完成最终计划中生电活动保护、队伍/区域权限、本地路径点显式导入、运行指标和三进程联调环境，并保持协议 2 与地图格式 2 不变。

## 修改内容

- 增加 TNT、爆炸、活塞、光照活动信号及可选 Carpet 更新抑制桥接。
- 增加 PUBLIC/TEAM 路径点、计分板队伍过滤、区域访问规则、审计日志和管理员命令。
- 增加 Xaero 本地路径点的当前维度读取、搜索、预览和确认上传。
- 增加全局/单玩家网络预算及 1,200 tick MSPT、任务耗时和带宽统计。
- 增加服务端、客户端 A、客户端 B 的独立 Loom 运行目录和本地验收脚本。
- 将预发布版本更新为 `2.1.0-beta.1`。

## 不包含的内容

- 不支持新的 Minecraft、Forge/NeoForge 或其他地图 Mod。
- 不改变同步协议号、地图格式号或固定 Xaero 版本。
- 不自动公开私人路径点，不强加载未加载区块，不上传世界存档或完整 NBT。
- 不包含生产发布或不可逆数据迁移。

## 风险等级

Medium

## 行为变化

修改前：活动保护主要依赖方块变化；路径点只有公共入口；没有本地导入、区域 ACL、全局带宽指标和隔离联调环境。

修改后：活动检测覆盖专用技术信号；路径点可按 PUBLIC/TEAM 共享并受区域 ACL 约束；本地路径点只能显式确认上传；管理员可审计、禁用和调整预算。

## 验证方式

- [x] 单元测试：102/102 通过
- [x] 集成构建：`gradlew.bat clean build`
- [x] 服务端冒烟：Fabric 服务端绑定 `127.0.0.1` 并进入 Done 状态
- [x] 安全测试：权限策略、区域访问和输入边界单元测试
- [ ] 双客户端人工端到端验收
- [ ] 大型 TNT/活塞/光照压力测试

## 评测结果

基线结果：此前构建通过；没有可比较的真实服务器负载基线。

修改后结果：25 个测试套件、102 个测试全部通过；服务端完整启动通过。

差异：新增活动、权限、导入和网络预算回归覆盖；真实 GUI/Xaero 和压力数据待人工采集。

## 性能和成本

响应时间：新增统计使用固定 1,200 tick 环形窗口；未采集真实 P95 基线。

Token 消耗：不涉及模型调用。

工具调用次数：路径点同步无新增自动调用；导入只在用户确认后发送一次创建请求。

## 安全影响

是否增加权限：否；管理员命令仍要求权限等级 2。

是否增加数据访问：客户端仅在用户打开导入页时读取当前维度的 Xaero 本地路径点。

是否增加网络访问：否。

是否涉及敏感信息：审计记录玩家 UUID、名称、计分板队伍和操作结果，不记录密钥或聊天内容。

## 发布方式

预发布版本；先在本地/测试服务器人工验收，再灰度部署。

## 回滚方式

停止服务端，回退本分支提交和 Mod JAR，并恢复同一时间点的 `<world>/xaero-mapsync_r/` 备份。客户端可清除本 Mod 的 revision/root 配置触发重新比较，不修改 Xaero 自身缓存。
